A chill startup plugin

Requires oh-my-zsh

Place in /Users/user/.oh-my-zsh/custom/plugins, in a parent folder named "anything" (I used 'communism'). Then, cd back to root and edit .zshrc to include your plugin. 

plugins=(... communism)
